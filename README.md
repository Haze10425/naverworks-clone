## 핵심 기능
- 스와이퍼 슬라이드 및 페이지네이션 구현
- 슬라이드 이전/다음 버튼(prev/next) 커스텀
- 돌아가는 숫자 룰렛 애니메이션 구현 (GSAP)
- 모바일/데스크탑 반응형 레이아웃 적용

---

## 기술적 특징
- 웹 표준과 접근성 고려
- CSS BEM 컨벤션을 활용한 구조화 및 유지보수 용이한 스타일링
- Swiper 옵션을 활용한 무한 루프, 반응형 슬라이드 구현
- GSAP ScrollTrigger와 matchMedia()를 이용한 반응형 애니메이션 적용
- 브라우저 창 크기 변경 시 addEventListener('resize')로 동적 높이 및 애니메이션 대응
